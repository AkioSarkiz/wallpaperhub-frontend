## Getting Started

- Clone the repository

```bash
git clone git@github.com:AkioSarkiz/wallpaperhub-frontend.git
cd wallpaper-frontend
```

- Install dependencies

```bash
pnpm install
```

- Create .env file and fill it in

```bash
cp .env.example .env
```

- Start the development server

```bash
pnpm dev
```
